How to Run Projet:
------------------

Requirements:
  1.Download and Install any local web server such as XAMPP/WAMP/MAMP/LAMP.
  2.Download the provided source code zip file 
		via google drive link: https://drive.google.com/file/d/1Tf3nw4zKKjl4NM81j-tBlDOicVSdiuW5/view
		or from attached file within email 
		or from github link: https://github.com/developerazadpro/xpeedStudio

Installation/Setup:
  1.Open your XAMPP/WAMP/MAMP/LAMP's Control Panel and start the Apache and MySQL.
  
  2.Extract the downloaded source code zip file.
  
  3.If you are using XAMPP, copy the extracted source code 'xpeedStudio' folder and paste it into the XAMPP's "htdocs" directory. 
    If you are using WAMP, paste it into the "www" directory. 
    If you are using MAMP, paste it into the MAMP's "htdocs" directory.
    If you are using LAMP, paste it into the LAMP's "www" directory.
	
  4.Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin
  
  5.Create a new database naming xpeed_studio
  
  6.Import the provided SQL file. The file is known as xpeed_studio.sql located inside the db folder of 'xpeedStudio'.
  
  7.Browse the Web Application in a browser. i.e. http://localhost/xpeedStudio


Features of this project:
  1. Table structure according to your requirements.
  2. Frontend validation.
  3. Backend validation.
  4. Form submission using jquery and ajax
  5. OOP and MVC pattern without any readymade PHP Frameworks.
  6. Login and registration system.
